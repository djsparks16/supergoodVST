# Obsidian Monster Glow-Up Pass

## Engine
- Added six new factory wavetable families: Growl, Formant, Metal, Reese, Acid, and Organ.
- Expanded table selector from 3 slots to 9 slots while preserving the User WAV import slot.
- Added oscillator warp modes Fold, Asym, and Quant.
- Preserved Osc A FM from Osc B and moved it to the expanded warp list as a dedicated final mode.
- Added selectable sub waveforms: Sine, Triangle, and Square.

## FX Rack
- Added Hyper: a stereo micro-delay widening stage for big Serum-style bass width.
- Added EQ: low/high tone shaping for faster bass sculpting.
- Added Limit: final soft-ceiling limiter to keep monster patches usable.
- Existing Womp, Crush, Distortion, Chorus, Phaser, Delay, Reverb, Width, Compressor remain intact.

## UI
- Larger default editor: 1280 x 920.
- Animated neon bloom field behind the panels.
- Added Sub waveform selector to the Sub/Noise card.
- Added Hyper, EQ, and Limit modules to the FX Rack card.
- Updated header status strip to reflect the expanded bass bank and WT+ engine.

## Presets
- Added 30 new bass patches in `Presets/`.
- Normalised all original presets with the new parameters for forward compatibility.
- Total preset count: 35.

## Notes
This is still not a literal clone of any commercial synth. It is a major feature pass toward comparable wavetable-synth breadth: more source spectra, more warp behaviour, deeper bass FX, and a usable patch bank.
